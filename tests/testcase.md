## Count Unique

| Test case           | Case   | Return |
| ---                 | ---    | ---      |
| None    | None | TypeError |
| Empty list | [ ]  | 0  |
| Multiple items, many times  |  ['a', 'b', 'b', 'b', 'a', 'c', 'c']  | 3 |
